# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities.

Use GitHub's private vulnerability reporting:
https://github.com/garymike/unifi-security-advisor/security/advisories/new

This is a personal project maintained on a best-effort basis. I'll try to acknowledge reports within roughly 48 hours and to look into confirmed issues as soon as I reasonably can — but these are goals, not guarantees, and response and resolution times may vary.
